## About

**Mugen Art Lab** is an independent personal project focused on creating practical tools for streaming, audio, media processing, and creative workflows.

These tools began as solutions to real problems in my own setup and are shared publicly in the hope that they may help others too.

## Projects

### Mugen Deej

A portable bilingual Windows client for deej-compatible USB audio controllers.

The first public release is currently being prepared.
